# Múltiplos arquivos PDF por acordo

- O seletor permite escolher vários PDFs de uma vez (`multiple`).
- Novas seleções são acumuladas; não substituem as anteriores.
- Cada arquivo pode ter até 10 MB.
- Os arquivos selecionados podem ser exibidos ou removidos antes de salvar.
- O ícone isolado de PDF foi substituído pelo botão **Arquivos**.
- O botão abre uma lista com todos os anexos e uma ação **Exibir** para cada item.
- PDFs antigos continuam disponíveis por compatibilidade.
- Inclui a migration `20260722150000_MultiplosArquivosAcordo`.
