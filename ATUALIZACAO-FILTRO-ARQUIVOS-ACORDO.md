# Restrição de arquivos nos acordos

- Formatos aceitos: `.PDF`, `.DOCX`, `.JPG`, `.JPEG` e `.PNG`.
- A validação ocorre tanto no seletor do navegador quanto no código do servidor.
- Arquivos vazios ou com mais de 10 MB são recusados.
- PDF e imagens são abertos em uma nova guia; DOCX é baixado para abertura no editor compatível.
- Nenhuma migration foi adicionada.
