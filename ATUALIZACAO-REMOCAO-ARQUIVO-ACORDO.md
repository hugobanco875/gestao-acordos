# Remoção individual de arquivos do acordo

Na edição do acordo/processo, cada arquivo já anexado agora possui o botão **Remover**.

- A remoção exige confirmação.
- Somente o arquivo selecionado é marcado para exclusão.
- A exclusão é concluída ao salvar o acordo.
- Cancelar a edição mantém os arquivos no banco.
- Arquivos novos ainda não salvos podem ser removidos da seleção normalmente.
- Compatível também com o PDF legado armazenado na tabela `AcordosPdf`.
- Não exige migration.
