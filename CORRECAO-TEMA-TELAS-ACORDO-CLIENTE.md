# Correção de tema nas telas de cliente e acordo

Foram ajustados os links internos que abrem detalhes do cliente, novo acordo, edição do acordo e detalhes das parcelas para utilizarem navegação completa (`data-enhance-nav="false"`).

Isso garante que o tema selecionado seja aplicado antes da renderização dessas telas, mantendo o mesmo comportamento já validado nos menus principais.

Não há alteração de banco de dados ou migration.
