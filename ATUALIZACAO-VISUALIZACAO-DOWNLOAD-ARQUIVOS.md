# Visualização e download separados

- O botão **Exibir** apenas abre PDF ou imagem para visualização.
- A visualização não inicia download automaticamente.
- Foi adicionado um botão pequeno de download ao lado de **Exibir**.
- Arquivos DOCX informam que a visualização não é suportada pelo navegador e devem ser baixados pelo botão próprio.
- Alteração aplicada no menu de arquivos, detalhe do acordo e edição do acordo.
- Sem alteração de banco de dados ou migration.
